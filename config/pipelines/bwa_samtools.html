📖 Usage Guide

Activate the conda environment, then run: 

conda activate bioinf_bwa_samtools

# Step 1: bwa
bwa index /path/to/chr22.fa

# Step 2: bwa
bwa mem -t 4 -R '@RG\tID:SRR1517830\tSM:HG00096\tPL:ILLUMINA\tLB:exome' chr22.fa reads_R1.fastq.gz reads_R2.fastq.gz > aligned.sam

# Step 3: samtools
samtools sort -@ 4 -o aligned_sorted.bam aligned.sam

# Step 4: samtools
samtools index aligned_sorted.bam

# Step 5: samtools
samtools flagstat aligned_sorted.bam && samtools idxstats aligned_sorted.bam
